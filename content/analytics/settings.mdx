## General settings

Navigate to your tracker's **Settings -> General** page to update the tracker's name and domain.

### Name

The tracker name is a slug (lowercase letters, numbers, and hyphens). It's used to identify the tracker in the Dashboard.

### Domain

The domain tells Flowtag which website this tracker belongs to (e.g. `example.com`). Keep this in sync with the site where the tag is installed.

## Deleting a tracker

<Callout type="warn">
  Deleting a tracker permanently removes all associated session data. This action cannot be undone.
</Callout>

1. Go to **Settings -> General** for the tracker.
2. Scroll to the **Delete tracker** section.
3. Type the tracker name to confirm and click **Delete**.
