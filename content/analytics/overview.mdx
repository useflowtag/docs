## What is a tracker?

A **tracker** is a Flowtag analytics project tied to one website domain. Each tracker has a unique ID used by the JavaScript tag to attribute sessions to the right property.

Trackers collect:

- **Sessions** — open/closed visitor sessions with timestamps
- **Pages** — the paths visitors land on
- **Devices** — browser, OS, and device type
- **Geography** — country derived from IP (hashed for privacy)
- **Referrers** — where traffic came from
- **Tags** — custom metadata you attach to sessions

## Tracker status

| Status | Description |
|---|---|
| `active` | Collecting data normally. |
| `paused` | Tag is ignored; no new sessions are recorded. |
| `suspended` | Disabled by Flowtag for policy reasons. |

## Creating a tracker

1. In the Dashboard, go to **Trackers** and click **New tracker**.
2. Enter a **name** (slug format, e.g. `my-site`) and the **domain** of the website you are tracking.
3. Click **Create**.
